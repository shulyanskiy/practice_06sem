# 📡 Antenna Emulator (Эмулятор антенны)

Серверное приложение для эмуляции работы управляемой антенны. Программа прослушивает TCP-сокет, принимает текстовые команды и имитирует перемещение антенны по азимуту/углу места с сохранением текущего состояния.

## Задание (Цель проекта)

Реализовать программу-эмулятор антенны, которая:
1.  Прослушивает указанный сетевой порт (сокет).
2.  Принимает и обрабатывает команды управления.
3.  Поддерживает базовый набор команд для управления позицией.
